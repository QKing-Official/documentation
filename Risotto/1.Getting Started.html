Getting Started

1. Install Requirements

Risotto uses only the Python standard library — no external dependencies required. You just need Python 3.8+ installed.

2. Prepare Configuration

Project Structure

your-project/
├── docs/
│   ├── index.md
│   ├── category1/
│   │   ├── intro.md
│   │   └── usage.md
│   └── category2/
│       └── guide.md
├── example.config.risotto
└── risotto.py

Copy the example configuration to your working config file:

cp example.config.risotto config.risotto

You can edit config.risotto to customize:

  • Site name and title
  • Description and favicon
  • Logo for light/dark themes
  • Theme colors
  • Output directory
  • Home page name
